# CareerMate (Prototype)

## 📌 Description
CareerMate is a web-based career guidance platform designed to help students make informed decisions about their future. It provides personalized suggestions for courses, colleges, and jobs, along with AI-based assistance.

---

## 🚀 Features

- 🤖 AI Assistant to answer career-related questions  
- 🎓 Course suggestions after 10th & 12th  
- 🏫 College recommendations based on cutoff & region  
- 🗺️ Map view with pinned college locations  
- 💼 Job suggestions based on user interests  
- 📄 Automatic resume generation  
- 📊 Identifies weaknesses and suggests improvement tips  
- 📩 SMS reminders for counselling dates and entrance exams
